[[query-dsl-or-query]]
=== Or Query

deprecated[2.0.0-beta1, Use the `bool` query instead]

A query that matches documents using the `OR` boolean operator on other
queries.

[source,js]
--------------------------------------------------
{
    "filtered" : {
        "query" : {
            "term" : { "name.first" : "shay" }
        },
        "filter" : {
            "or" : [
                {
                    "term" : { "name.second" : "banon" }
                },
                {
                    "term" : { "name.nick" : "kimchy" }
                }
            ]
        }
    }
}
--------------------------------------------------
